❤️ Heart Stroke Prediction

A Machine Learning web application built with **Streamlit** that predicts the risk of heart stroke using a trained **K-Nearest Neighbors (KNN)** model.

## 🚀 Features

- Real-time heart stroke prediction
- Interactive Streamlit interface
- KNN Machine Learning model
- Automatic data preprocessing using StandardScaler
- Simple and user-friendly design

## 🛠️ Tech Stack

- Python
- Streamlit
- Pandas
- Scikit-learn
- Joblib

## ▶️ Run Locally

1. Clone the repository
```bash
git clone https://github.com/yourusername/Heart-Stroke-Prediction.git
```

2. Install dependencies
```bash
pip install -r requirements.txt
```

3. Run the app
```bash
streamlit run app.py
```
